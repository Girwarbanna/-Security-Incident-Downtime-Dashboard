This repository contains a fully interactive **Security Incident & Downtime Analysis Dashboard** created using **Power BI**.  
The dashboard provides real-time insights into incident response, SLA compliance, downtime patterns, and operational security performance.

> **Note:**  
> All data used in this project is **AI-generated sample data**.  
> No real or confidential organization data has been used.  
> Icons and images used in the report are also AI-generated.
